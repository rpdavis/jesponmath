# Long Division in JepsonMath - Quick Reference

## ✅ NEW! Easy Long Division

You can now create long division problems with a simple command!

## Basic Usage

**Type this:**
```
$\longdiv{45}{345}$
```

**You get:** 45)‾345 (proper long division bracket)

## Quick Examples

### Example 1: Simple Division
```
Solve: $\longdiv{12}{144}$
```

### Example 2: Larger Numbers  
```
Calculate: $\longdiv{23}{552}$
```

### Example 3: For Display (Centered, Bigger)
```
$$\longdiv{123}{15129}$$
```

## Alternative: Simpler Syntax

If you prefer a shorter command:
```
$\ldiv{8}{96}$
```

## Showing Work (Step-by-Step)

For problems where students show their work:

```
$$\begin{array}{r}
  \phantom{0}12 \\
8\overline{)96} \\
  -8\phantom{0} \\ \hline
  \phantom{0}16 \\
  -16 \\ \hline
  \phantom{0}0
\end{array}$$
```

This shows:
```
     12
8)‾96
  -8
  ---
   16
  -16
  ---
    0
```

## Tips

- Use `$...$` for inline math (in a sentence)
- Use `$$...$$` for display math (centered, on its own line)
- The first number in `\longdiv` is the **divisor** (what you divide by)
- The second number is the **dividend** (what you're dividing)

## Common Patterns

**Division with remainder:**
```
$$\begin{array}{r}
  7\text{ R}2 \\
5\overline{)37} \\
  -35 \\ \hline
  \phantom{0}2
\end{array}$$
```

**Decimal division:**
```
$$\begin{array}{r}
  2.5 \\
4\overline{)10.0} \\
  -8\phantom{.0} \\ \hline
  \phantom{0}2.0 \\
  -2.0 \\ \hline
  \phantom{0}0
\end{array}$$
```
